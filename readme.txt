== Installation ==

Pour installer le plugin: <br><br>
- Décompresser le fichier avec l'extension ZIP, <br>
- Placer le dossier décompréssé dans le répertoire "wp-content/plugins/" de votre site internet, <br>
- Dans l'administration de votre site, aller dans la section "Extensions" et cliquer sur "Activer" pour le plugin "Réservation Facile", <br>
- Un nouveau menu "Réservation Facile" apparaît alors. <br>
